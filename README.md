# 에픽세븐 비상런 자동화 한글 버전

이 프로젝트는 Solunium(https://github.com/Solunium/Epic-Seven-E7-Secret-Shop-Refresh)이 
제작한 파이썬 기반의 에픽세븐(Epic Seven) 비밀 상점 매크로 프로그램을 한글화하고, 일부 로직을 개선해 속도를 조금 더 빠르게 제작한 프로그램입니다.

원작자의 라이센스 조항에 따라  GPL-3.0 license로 운영 됩니다.

## ✨ 사용 방법
1단계: 최신 릴리스를 다운로드하세요. 반드시 (Secret_Shop_KR.zip) 파일을 다운로드해야 합니다.

2단계: 다운로드한 압축 파일(zip)을 원하는 폴더에 압축 해제하세요.

3단계: 공식 클라이언트 또는 에뮬레이터에서 에픽세븐을 실행하세요. 

4단계: 실행 후 비밀 상점 화면으로 들어가세요. ** 오류 방지를 위해 꼭 비밀상점 페이지를 열어주세요 

4단계: 압축을 푼 폴더 안에 있는 Secret_Shop_KR.exe를 실행하세요.

5단계: 드롭다운 박스에서 사용 중인 클라이언트를 선택하세요.

6단계: 구매 시작 버튼을 누르세요



프로그램을 멈추려면 키보드의 ESC 키를 누르세요.

주의: 매크로를 실행 후 방치하기 전에, 프로그램이 성약의 책갈피와 신비의 메달을 제대로 인식하는지 반드시 확인하세요.

## 주의사항
프로그램 사용으로 발생하는 계정 제재나 손실에 대해서는 모두 사용자가 책임을 져야 합니다.
