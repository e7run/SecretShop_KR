# 에픽 세븐 비밀 상점 - 한글 버전
## 스크린샷
![](https://e7run.github.io/SecretShop_KR/assets/se.png)

## 시작하기 전에
### 이 프로그램은
Solunium(https://github.com/Solunium/Epic-Seven-E7-Secret-Shop-Refresh)이 개발한 비상런 프로그램을 활용해 일부 로직을 변경하고, 한글화를 진행한 프로젝트입니다. 원작자의 라이센스 정책에 따라 GPL-3.0 license로 운영됩니다.

<br>

### 반드시 확인해주세요.
이 프로그램을 사용해서 발생하는 계정 제재나 재화의 손실은 사용자 책임입니다.

### 사용방법
1. 공식 클라이언트 또는 에뮬레이터로 에픽세븐을 실행합니다.
2. 에픽세븐에서 비밀 상점으로 진입합니다. (이정표 등 안전성을 위해 반드시 비밀 상점 페이지에서 실행하세요)
3. Secret_Shop_KR.zip을 다운로드 받은 후 Secret_Shop_KR.exe를 실행합니다.
4. 구매 시작을 누르면 동작합니다.
